Installation Instructions:
  1. Download the zip file.
  2. Extract the zip file (some browsers may do this automatically).
  3. Double-click to open the app. If you get a security error (GateKeeper), you can try right clicking, and selecting Open, which should give you an additional "Open" option in the popup.
  4. Either add your BIOS files to '[home directory]/Library/Application Support/AetherSX2/bios', or change the BIOS path in settings.
  5. Add a directory from the settings menu to scan for games. Don't select your home directory or the root of any external volumes, it'll take forever. Pick the directory with your game dumps.
  6. Launch game and enjoy.
